SocialSite is a social networking server and a set of gadgets that make it possible to add social networking features to existing web sites The server is based on Apache Shindig, written in Java and adds a complete persistent social graph (stored via JPA SocailSite's user interface is an Admin Console and a set of OpenSocial Gadgets that provide a complete social networking interface.The project has been incubating since April 30, 2009. The SVN space, mailing lists and initial user accounts have been created but we are still waiting for Sun to sign-off on the code grant. It's been about six months since Sun told us this would happen, we've been checking in with Sun every couple of weeks and apparently we're getting close.Signed off by: Dave Johnson.
