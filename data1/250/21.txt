Wookie is a Java server application that allows you to upload and deploy widgets for your applications. Wookie is based on the W3C Widgets specification, but widgets can also be included that use extended APIs such as Google Wave Gadgets and OpenSocial.Wookie has been incubating since 17th July 2009.Since February 2011:. Release Candidate 0.9 put out for testing and feedback from wookie-dev members. Feedback and issues for Release Candidate 0.9 posted to JIRA. Almost all issues closed, new RC build imminent.
