Kato was accepted into the Incubator on 6 November 2008.Kato is a project to develop the Specification, Reference Implementation, and TCK for JSR 326: the JVM Post-mortem Diagnostics API.Recent Activity:.A JVMTI agent has been contributed by Paul Sobek along with an implementation of the API.The JSR specification document has been written and is building.The JSR specification has been submitted to the JCP as an EDR.The project is being built on the ASF's hudson server.The following is planned for next reporting period:.Our first release of the RI, demos and TCK.Before this project can be graduated we need to produce a usable implementation of the API and more useful tools to encourage adoption and participation of a much needed community.Signed off by: rdonkin.
