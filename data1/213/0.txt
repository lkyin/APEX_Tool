iPMC Reviewers: wrowe, bdelacretaz, yoavs, jimSling is a framework to develop content centric web applications based on the idea of modularizing the rendering of HTTP resources. Sling entered incubation on September 5th, 2007.The Sling project has just started. The basic infrastructure (mailing lists, subversion, issue tracker, web site) is mostly in place; the only things still missing are one committer account and a wiki space. The software grant for Sling has been received and the initial source code is now available in svn.iPMC Comments:.Looking forward to a big picture of how Sling will move itself through the.process to graduation, but we don't expect that quite yet.
